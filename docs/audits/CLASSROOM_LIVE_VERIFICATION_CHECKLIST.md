## 2) لماذا التحقّق الحيّ محجوب
- مسار `/classroom/[classId]` (Server Component) يفشل-مغلق إن لم يوجد: **فصل حقيقي** + **تكليف المعلّم لهذا الفصل** (`teacher_assignments`) + **طلاب في الفصل**. كلّها = 0 في DB الحية.
- لا توجد آلية seed/demo معتمدة لإنشاء مدرسة/فصل/طلاب تجريبيين (لا `db/seed`، لا `supabase/seed`، لا npm script). الموجود: `scripts/init-system-owner.mjs` (مالك نظام) + M81 (مؤشّر جودة حقيقي) — لا شيء منها ينشئ فصلاً/طلاباً.
- إنشاء فصل/طلاب/تكليفات مباشرةً في المدرسة الحيّة الوحيدة (الفلاح) = **بيانات إنتاجية وهمية** (ممنوع صراحةً).
- لا تتوفّر بيانات اعتماد دخول معلّم لجلسة متصفّح مُصادَقة (preview tools تتطلب جلسة مُصادَقة بدور teacher + school_id في JWT).

## 3) البيانات المطلوبة لإتمام التحقّق الحيّ (تُوسَم demo/test بوضوح)
1. **فصل** (`classes`) في مدرسة اختبار: `school_id` + `grade_level` + `section` + `stage_id` + `academic_year_id`.
2. **تكليف معلّم** (`teacher_assignments`): `teacher_id` = `user_id` لبيرسونا المعلّم + `class_id` + `subject_id` + `academic_year_id` + `school_id`.
3. **طلاب** (`student_profiles`): `class_id` = الفصل · `school_id` = المدرسة (عدد صغير، 3–5).
4. **جلسة معلّم مُصادَقة**: JWT بـ `app_metadata.role='teacher'` و`school_id` للمدرسة (بيانات اعتماد اختبار من المالك).
5. (لمنح المكافآت) أدوار المشغّل: `teacher/school_admin/school_principal/activity_leader`.

## 4) المسارات والنتائج المتوقَّعة (عند توفّر بيانات الاختبار)
| السطح | الخطوات | النتيجة المتوقَّعة |
|---|---|---|
| توجيه | `/classroom` → ظهور الفصول المُسنَدة → نقر فصل | `/classroom/[classId]` يفتح (لا `notFound`) |
| حفظ المقاعد | عرض «خريطة» → «وضع الترتيب» → اضغط طالبين → «حفظ المقاعد» | «✅ تم حفظ مخطّط المقاعد» + صفّ في `classroom_metadata.seating_map` |
| حفظ الأدوار | اختر طالباً واحداً → رقاقة دور → «حفظ الأدوار» | «✅ تم حفظ أدوار الطلاب» + `classroom_metadata.student_roles` |
| منح نجوم | اختر طلاباً → StarSelector (3) → حفظ | صفوف `classroom_rewards` (reward_type=star) + تحديث «ملخّص المكافآت» |
| سلوك إيجابي | اختر طلاباً → زر «شارك اليوم» مثلاً | `classroom_rewards` (positive_point) |
| منح وسام | اختر طالباً → BadgesModal → اختر وساماً | `classroom_rewards` (badge) + عدّاد وسام على بطاقة المقعد |
| عرض الأوسمة | عرض «خريطة» | عدّاد الأوسمة + tooltip بأسماء الأوسمة على بطاقة الطالب |
| سجلّ المكافآت | اختر طالباً واحداً | قسم «سجل المكافآت والأوسمة» يعرض الصفوف الحقيقية؛ وعند عدمها: «لا توجد مكافآت أو أوسمة مسجلة لهذا الطالب بعد.» |

## 5) كيفية رفع الحجب
- **(أ)** اعتماد مدرسة/معلّم اختبار + seed موسوم demo/test (لا في مدرسة إنتاج)، أو
- **(ب)** تزويد المالك ببيانات اعتماد معلّم اختبار + فصل تجريبي بطلاب، ثم تشغيل فحوص `preview_*` غير المُدمِّرة.
- حتى ذلك الحين: التحقّق تمّ عبر `npm run build` + `npx tsc --noEmit` + `npm run lint` + `npm test` فقط (لا تحقّق متصفّح حيّ).
